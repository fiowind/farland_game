# farland-ui

#### Install dependencies

```shell
npm install
```

#### Running locally

`npm run dev`

#### Deploy

`npm run build`

`npm start`


1. 开通ecs
2. 安装node和mongo
3. 后台运行mongo
4. clone 这个库
5. npm i 安装依赖
6. 配置server.js mongo地址
7. npm run build 编译
8. 启动：npm i pm2 -g && pm2 start pm2.js
9. 配置端口开放
10. 域名指向当前ip地址
